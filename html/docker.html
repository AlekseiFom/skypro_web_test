Docker — команды

Работа с контейнерами

  • docker run – запуск – Запуск контейнера на основе образа
  • docker ps – список контейнеров – Показать все работающие контейнеры
  • docker stop – остановка – Остановка контейнера

Образы

  • docker build – сборка – Создание образа из Dockerfile
  • docker pull – скачать – Скачивание образа из репозитория
  • docker push – отправить – Отправка образа в удалённый репозиторий
